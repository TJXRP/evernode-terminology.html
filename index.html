Evernode Terminology

Evernode.Forum | Discord | Telegram
Autopilotautomates processes

Autopilot: A system on Evernode that automates certain processes, reducing the need for manual intervention in network operations.

Codiushosts smart contracts

Codius: An earlier project similar to Evernode that aimed to host smart contracts.

DAppdecentralized app

DApp: A decentralized app that runs on a blockchain network without a central server. DApps can be built on various blockchains (like Ethereum or Evernode).

Decentralizationdistributed control

Decentralization: No one controls the entire network. Many computers (nodes) share the work.

DeFifinancial services on blockchain

DeFi: Financial services like lending and trading built on Evernode, without traditional banks.

DePINphysical infrastructure network

DePIN: Decentralized Physical Infrastructure Network—enabling services like DeFi and other decentralized apps to interact with real-world assets.

Edge Computingdistributed nodes for performance

Edge Computing: Using many Evernode nodes spread out to make apps faster and more reliable.

EVREvernode's token

EVR: Evernode's token used to pay for services and hosting on the network.

Evernode 1.0full version release

Evernode 1.0: The first full version of Evernode that’s ready for real-world use.

EVM Compatibilityruns Ethereum contracts

EVM Compatibility: Evernode supports the Ethereum Virtual Machine, allowing developers to use Ethereum-based smart contracts seamlessly.

Halvingreduces mining rewards

Halving: A scheduled event where the reward for mining or validating transactions is cut in half, reducing new token supply (common in networks like Bitcoin).

Hostinghosting apps

Hosting: Running apps or websites on Evernode’s decentralized network.

Hookstriggers for actions

Hooks: Small bits of code that automatically trigger actions like running smart contracts.

Hot Pocketruns app instances

Hot Pocket: A slot in an Evernode node with support for Docker deployments, WebSocket, TCP, and UDP connections. It ensures apps run efficiently in a secure environment.

Hot Pocket Clustergroup of hot pockets

Hot Pocket Cluster: A group of two or more Hot Pockets working together, with an optional consensus engine that executes actions based on agreed-upon inputs.

Instanceseparate app environment

Instance: A separate environment where Evernode apps run.

Interoperabilityworks with other chains

Interoperability: Evernode can talk and work with other blockchains easily.

Nodescomputers running Evernode

Nodes: Computers that help run Evernode by hosting apps and keeping the network running.

Multichainsupport for multiple blockchains

Multichain: Evernode’s ability to support apps across multiple blockchain networks.

Scalabilityhandles increasing work

Scalability: Evernode can handle increasing amounts of work efficiently as more nodes are added to the network.

Sashimonomanages nodes

Sashimono: The system that manages Evernode nodes and launches Hot Pockets/instances, enabling smooth operation of decentralized services.

Serviceuseful offerings

Service: Something useful offered on Evernode, like file storage or hosting apps.

Smart Contractsself-executing agreements

Smart Contracts: Programs that automatically execute agreements when certain conditions are met.

Trustlineconnection between accounts

Trustline: A connection between two accounts on the XRP Ledger that allows them to hold and transact in tokens other than XRP.

Tenant Feefee for services

Tenant Fee: The fee required to execute services or smart contracts on Evernode, paid in EVR.

Web Socketreal-time communication

Web Socket: A communication protocol Evernode uses to allow apps to communicate with servers in real-time.

xAppapp for Xaman Wallet

xApp: An app built specifically for the Xaman Wallet. While it communicates with the Xahau and XRP Ledger (XRPL), it can utilize Evernode.

Xahau NetworkEvernode's blockchain

Xahau Network: The blockchain Evernode uses for its EVR token and value transfers. Evernode is a stand-alone network!

XWCode IDEcloud IDE for smart contracts

XWCode IDE: A cloud-based Integrated Development Environment (IDE) for writing and deploying smart contracts in any coding language on Evernode.
